#desvirt
DES-Virt is the DES Testbed virtualization framework.

Dependencies
------------
When running under Debian squeeze, des-virt needs the following packages:

		kvm 
		libvirt-bin 
		python-libvirt 
		vlan 
		bridge-utils 
		ebtables 
		uml-utilities
